# Node Environment

Node.js docker environment.

## Requirements

* [Docker](https://www.docker.com/) installed and running.

## Building

Builds the docker container.

```bash
  $ ./build
```

## Running

* The `./data` directory is mapped to `/data` inside the container.
* Exposes ports `1234` and `11234`.

```bash
  $ ./run
```
